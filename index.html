COMSOL Video Gallery

Information Architecture | Design | Development

The objective of this project was to re-design the existing video gallery for COMSOL. The focus was to improve the user experience and the visual design of the user interface. A more robust navigation and filtering were implemented to better assist the user in their search for content. The minimalist design and muted color scheme put more focus on the rich colors of the videos and thumbnails. The responsive layout enabled users to easily browse and view the content from any device.

COMSOL Conference 2015

Information Architecture | Design | Development

The objective of this project was to re-design the existing conference site for COMSOL. The focus was on improving the visual design of the user interface and implementing a responsive layout. The home page of the conference was designed to emphasize the multitude of conferences available internationally for the COMSOL user. User experience was a large focus with this re-design, making sure the user was able to access information and register for the appropriate conference.
